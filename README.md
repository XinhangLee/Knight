## 游戏元素

### 人物

> <img src="rsc/hero/Hero.png" width="64" height="64" alt="Hero" title="Hero">

```c++
class Hero {
    int HP;
    int Shield;
    int Energy;
    int Speed;
    Weapon weapon;
    Postion Hero_pos;
public:
    Hero();
    ~Hero();
    void Move();
    void Attack();
}
```
```c++
struct Postion {
    double x;
    double y;
}
```

class Monster {
